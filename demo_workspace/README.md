# Demo Workspace

This sanitized workspace lets reviewers run Course Materials Ops Agent without
access to the author's private course notes, browser sessions, or credentials.

The real project uses the same file structure in a local course workspace:

- `notes/link-log.md`
- `notes/course-log.md`
- `work/day-*/material-notes.md`
- `work/day-05/production-checklist.md`
- `capstone/*.md`
- `submissions/final-assignment-status.md`

No secrets, API keys, emails, cookies, or private account data are included.
